Railmail jest zaprojektowany do obsługi z klawiatury. Dwie rzeczy do zapamiętania: **⌘K** otwiera paletę poleceń, a **⌘/** (lub **?**) pokazuje pełną listę skrótów wprost w aplikacji.

## Paleta poleceń (⌘K)

Naciśnij **⌘K** w dowolnym miejscu, aby otworzyć paletę, i zacznij pisać, aby wyszukać. Z palety możesz:

- **Tworzyć**: nowa kampania (**C**), kampania na pełnym ekranie (**F**), nowy temat (**T**).
- **Nawigować**: przejdź do Aktywności, Kampanii, Subskrybentów, Tematów, Segmentów, Automatyzacji lub Raportów, albo wyszukaj i otwórz konkretny temat.
- **Ustawienia**: otwórz Preferencje, Profil, Ustawienia projektu lub Ustawienia domeny.
- **Akcje zbiorcze**: gdy na liście są zaznaczone wiersze, pasujące akcje pojawiają się też tutaj.
- **Wylogować się**.

## Globalne

| Skrót | Akcja |
| --- | --- |
| ⌘K | Otwórz paletę poleceń. |
| ⌘/ lub ? | Pokaż listę skrótów klawiszowych. |
| [ | Przełącz pasek boczny. |
| / | Fokus na wyszukiwanie. |
| ⌘N | Utwórz (zależnie od miejsca, w którym jesteś). |

## Nawigacja „idź do" (naciśnij G, potem literę)

| Sekwencja | Przechodzi do |
| --- | --- |
| G D | Pulpit |
| G C | Kampanie |
| G T | Tematy |
| G S | Subskrybenci |
| G A | Automatyzacje |
| G E | Segmenty |
| G L | Strumień aktywności |
| G R | Raporty |
| G F | Pola niestandardowe |
| G M | Członkowie |
| G X | Ustawienia |

## Listy (kampanie, subskrybenci, tematy, segmenty, automatyzacje)

| Skrót | Akcja |
| --- | --- |
| J / K lub strzałki | W dół / w górę. |
| Enter | Otwórz zaznaczony wiersz. |
| X | Zaznacz wiersz. |
| ⌘A | Zaznacz wszystko. |
| E | Edytuj. |
| D | Duplikuj. |
| ⌘⌫ | Usuń. |
| Esc | Wyczyść zaznaczenie. |
